Random Colors 1.0.0

The app is a simple random color generator.

- Click on the central frame in the program window (initially gray)
- A random color will appear in the frame
- The textbox below the color frame will display RGB and HEX values of the color
- You can copy the HEX code and paste it into an image editing app, a vector graphic app, or
   any other kind of visual art and communication application
- Click again on the frame to generate a new color
